# Contributing

We love contributions! This guide will help you get started.

## Bug Reports

If you find a bug, please [open an issue](https://github.com/natestott/alle/issues) and provide as much detail as possible.

## Feature Requests

If you have an idea for a new feature, please [open an issue](https://github.com/natestott/alle/issues) to discuss it.

## Pull Requests

1. Fork the repo and create a new branch.
2. Make your changes and run `bun run validate` to ensure all checks pass.
3. Submit a pull request.

*This guide is a work in progress.*
