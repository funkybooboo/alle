# Packages

This directory contains the source code for the Alle application, which is organized into a monorepo with the following packages:

*   [`client`](./client) — The React frontend.
*   [`server`](./server) — The Bun API server.
*   [`shared`](./shared) — Code that is shared between the client and server.

To learn more about the project's architecture, see the [**Architecture**](../docs/03-architecture.md) documentation.
