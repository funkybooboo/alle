# Client

This is the SvelteKit frontend for the Alle application. It runs on port 3000 and communicates with the server at port 4000.

## Quick Start

```bash
mise run client
```

The client will be available at `http://localhost:3000`.

To run both client and server together:

```bash
mise run dev
```
